O que é protocolo HTTP?

Protocolo HTTP é Protocolo de de transferencia de Hipertexto. Estrutura da informação, é o que é quando solicitada ao servidor, e quando a resposta volta.

Descreva o Protocolo HTTPS ? 

Protocolo HTTPs é o Protocolo de transferencia de hipertexto seguro. ele é basicamente a mesma coisa que o HTTP, porém é utilizado quando precisa acessar paginas criptografadas. Portanto definido como mais seguro, receberá informações mais seguras do servidor. 

Qual é a principal diferença entre os Protocolos HTTP e HTTPS?

HTTP nao consegue acessar paginas web em seu servidor que seja criptografado.
HTTPS já consegue acessar paginas web criptografadas.

O que é uma HTTP Request (Requisição HTTP)?

Request é quando voce solicita algo do servidor, eu digito um site e solicito um retorno do servidor com a pagina web.

O que é uma HTTP Response (Resposta HTTP)?

É a resposta do servidor com a pagina web solicitada.

Quais são os tipos de Requisições HTTP (Verbos), mais utilizados no Desenvolvimento WEB?

GET - Usado para coletar informações
POST - Usado para colocar informações
PUT - Usado para substituir informações
DELETE - USado para deletar informações.

Quais são os tipos de Respostas HTTP (Status code), mais utilizados no Desenvolvimento WEB?

Codigos de erros de Clliete 400 a 499 
Erros de servidor 500 a 599
